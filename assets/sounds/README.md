# Reminder Sounds

This folder should contain the following audio files (MP3 format):

- `default.mp3` - Default notification sound
- `soft_chime.mp3` - Soft chime notification
- `bright_bell.mp3` - Bright bell notification  
- `gentle_wave.mp3` - Gentle wave notification

## Adding Sound Files

You can:
1. Record/create your own notification sounds
2. Download royalty-free notification sounds from sites like:
   - freesound.org
   - zapsplat.com
   - soundbible.com

Ensure audio files are:
- MP3 or WAV format
- Short duration (1-3 seconds recommended)
- Appropriate volume for notifications
- Named exactly as shown above
